Koa for Trails.JS (via Trailpack)

## Install

```sh
$ npm install --save trailpack-koa
```

## Configure

```js
// config/main.js
module.exports = {
  packs: [
    // ... other trailpacks
    require('trailpack-koa')
  ]
}
```
